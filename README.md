## Project Overview

Cardiopulmonary sounds include heart and lung sounds that are commonly recorded together using a stethoscope. These sounds may overlap, making separation difficult using traditional signal processing methods.

This project focuses on studying and documenting a machine learning-based approach for separating mixed cardiopulmonary audio signals into two outputs:

1. Heart sound signal
2. Lung sound signal

The project is developed as a Software Engineering application-based prototype/proof of concept.

## Project Scope

The project focuses on:

- Cardiopulmonary sound separation
- Heart and lung sound separation
- Machine learning-based audio separation
- Audio preprocessing
- Feature extraction
- Public datasets
- System design
- Evaluation metrics

The project does **not** focus on disease detection or medical diagnosis.

## FYP Objectives

The objectives of this project are:

1. To study existing techniques for cardiopulmonary sound separation.
2. To design and develop a machine learning-based system for sound separation.
3. To implement and test the system using real audio datasets.
4. To evaluate the performance of the system using suitable metrics.
